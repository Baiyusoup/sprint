## 原因
1. 网络延时问题
2. 资源文件体积过大
3. 资源重复请求去加载了
4. 脚本阻塞渲染


## 解决方案

### 页面渲染优化
1. js和css外链的位置
2. 合理的HTML结构
3. 减少重排和重绘
4. 降低css选择器的复杂性


### 资源加载优化
1. 压缩代码
2. 服务器启用gzip压缩
3. 图片压缩
4. 代码拆分


减少http请求次数
1. http缓存
2. 雪碧图

提高http请求响应速度
1. cdn
2. dnd prefetch
3. http2


资源加载时机
1. 按需加载
2. 懒加载
3. 预加载
