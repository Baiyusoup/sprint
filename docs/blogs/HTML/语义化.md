## 标签
- header 用于展示介绍性内容，比如logo、搜索框、作者名称等等。
- nav 用于当前文档或其他文档中提供导航链接
- section
- article 文档中可独立分配的或可复用的结构，比如一个帖子、一篇文章等等
- aside
- footer
- main

## 作用
标签名传递了它这一块内容的一些重要信息，让人能够一眼望去有个大概的了解。
作用：
1. 有利于SEO（搜索引擎优化）
   1. title、descript、keywords
   2. html标签方式的使用
2. 便于团队开发和维护
3. 有利于设备解析
