## 柯里化
柯里化Currying式把接受多个参数的函数变换成接受一个单一参数的函数，并且返回接受余下参数且返回结果的函数的技术。

```javascript
function curry(fn, arity = fn.length, ...args) {
  if (arity <= args.length) {
    return fn(...args);
  }
  return curry.bind(this, fn, arity, ...args);
}
```

## 组合函数
执行从右到左的函数组合

```javascript
const compose = (...fns) => fns.reduce((f, g) => (...args) => f(g(...args)));
```
